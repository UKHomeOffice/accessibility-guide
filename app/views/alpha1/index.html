Designing and building accessible services ensures everyone can use your service.

Principles of building accessible services

If any of these are not met, users with access needs will not be able to use your service.

Users need to be aware of and be able to perceive all content and UI elements

Users need to be able to interact with and use all UI elements

Users need to be able to understand the service and it's content

Users need to be able to access a service with any technology, both old and new
