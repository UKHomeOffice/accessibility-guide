Home Office accessibility guide

Services must be accessible to everyone. We need to design and build services that match the way that people use them.

Accessibility principles

If any of these are not met, users with access needs will not be able to use your service.

Users need to be aware of and be able to perceive all content and user interface

Users need to be able to interact with and use the user interface

Users need to be able to understand the service and it's content

Users need to be able to access a service with any technology, both old and new

Basic accessibility checks

Anyone should be able to make these basic accesibility checks. Once done, you can contact the accessibility team for a more comprehensive audit.

Colour

  • Don't use only colour as the only means of conveying information to a user
    • Examples: Buttons, labels, graph data

Content

  • Supply an easy to find link back to the start of the service
  • Text is left aligned
  • Text is no longer then 75 characters per line
  • Text is styled normally, not in bold, italics or all caps
  • Text is styled normally, not in bold, italics or all caps
  • Each link describes where it goes
    • Examples:
  • Text contrasts clearly against the background
    • Examples: black text on a white background
  • Content is easy to read, written for a minimum reading age of 11
  • Content does not use abbreviations or acronyms
  • Use headings to break up content

Title

The title of a page can be found in the browser tab.

  • The title of the page describes the main purpose of the page
  • The title and the main heading on the page are the same or similar

Forms

  • Every form field has a label
  • Selecting a form field causes the field to be focused

Using the keyboard

To use the keyboard, press the Tab key to move to the next focusable element on the page and Shift+Tab keys to move to the previous element. Use either the Spacebar or Enter to use links and buttons.

  • The currently focused element is clearly shown
  • The order elements become focused is sensible
  • The page is easy to navigate
  • A skip link appears at the beggining of the page
